Return data about a genre by name; http://localhost:8080/movies/:genreName GET JSON JSON
Return data about a director (bio, birth year, death year) by name; http://localhost:8080/directors/:directorName GET JSON JSON

Allow users to update their user info (username); http://localhost:8080/users/:id PUT JSON JSON
Allow users to add a movie to their list of favorites; http://localhost:8080/users/:id/:movieTitle POST JSON text
Allow users to remove a movie from their list of favorites; http://localhost:8080/users/:id/:movieTitle DELETE JSON text
Allow existing users to deregister. http://localhost:8080/users/:id DELETE JSON text
